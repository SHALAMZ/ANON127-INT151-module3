รู้ความต่างของ Synchronous vs Asynchronous ว่าทำไม setTimeout ถึงไม่รอ และโค้ดมันรันข้ามไปก่อน
เข้าใจว่า Promise ไม่ได้ให้ค่าทันที แต่ให้ object ที่ยัง pending ต้องจัดการเอง
เรียนรู้วิธี resolve / reject เพื่อกำหนดผลลัพธ์สำเร็จหรือผิดพลาด
เห็นว่าถ้า console.log(promise) จะยังไม่ใช่ค่าจริง ต้องใช้ .then() หรือ await
ได้รู้ 2 วิธีจัดการงาน async แบบถูกต้อง:
1) .then().catch()
2) async / await + try/catch
เข้าใจ flow ของ event loop แบบพื้นฐาน ว่า async code จะถูกดันไปหลังสุด
